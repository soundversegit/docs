## Extend Music using Audio URL

> This API allows you to **extend an existing song** by providing a **publicly accessible URL** of the original audio.  
Choose between:


- üéß **HD Quality** ‚Üí `sansaar_1` (Higher fidelity, slower processing)
- ‚ö° **Fast Mode** ‚Üí `sansaar_1_lite` (Quicker processing, lower fidelity)


---

#### **Core Parameters**

> üõà The parameters below define the request body for this API.

| Parameter      | Type     | Required?   | Description                                                                    |
| -------------- | -------- | ----------- | ------------------------------------------------------------------------------ |
| **`userId`**   | `string` | ‚úÖ Required | &#x2003;&emsp;&emsp;Unique ID of the user.                                     |
| **`audioUrl`** | `string` | ‚úÖ Required | &#x2003;&emsp;&emsp;Publicly available URL of the audio file to extend.        |
| **`prompt`**   | `string` | ‚úÖ Required | &#x2003;&emsp;&emsp;Text description guiding how the music should be extended. |
| **`model`**    | `string` | ‚ö†Ô∏è Optional | &#x2003;&emsp;`"sansaar_1"` (HQ) or `"sansaar_1_lite"` (Faster, default).      |

#### Code Samples

<Tabs>
  <Tab title="Python">
  
  ```python
    import requests

    url = "https://api.soundverse.ai/generate/extend"
    headers = {
        "Authorization": "Bearer your_api_key_here",
        "Content-Type": "application/json"
    }
    payload = {
    userId: "67b9ca2f20e711017fc1b6c5"
        "audio_url": "
    https://storage.googleapis.com/x-one/67b9ca2f20e711017fc1b6c5/67b9ca2f20e711017fc1b6c5/xone20250312100522scsuxkec.wav
    ,
        "prompt": "Extend the provided audio clip by generating an additional 45 seconds of music. Maintain the style and mood of the original piece while seamlessly continuing the composition.",
        "model": "sansaar_1_lite"
    }

    response = requests.post(url, json=payload, headers=headers)
    print(response.json())

    ```

   </Tab>
