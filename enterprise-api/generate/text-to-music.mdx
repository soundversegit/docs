## Generate Music from Text

This API allows you to generate **AI-powered music** based on a text description.  
Choose between:

- üéß **HD Quality** ‚Üí `sansaar_1` (Higher fidelity)
- ‚ö° **Fast Mode** ‚Üí `sansaar_1_lite` (Quicker processing)

---

#### **Core Parameters**

> üõà The parameters below define the request body for this API.

| Parameter      | Type      | Required?   | Description                                                      |
| -------------- | --------- | ----------- | ---------------------------------------------------------------- |
| **`userId`**   | `string`  | ‚úÖ Required | &#x2003;&emsp;&emsp;Unique ID of the user.                       |
| **`prompt`**   | `string`  | ‚úÖ Required | &#x2003;&emsp;&emsp;Text description of the music.               |
| **`duration`** | `integer` | ‚ö†Ô∏è Optional | &#x2003;&emsp;&emsp;Track length (`15s`, `30s`, `60s`, `180s`).  |
| **`model`**    | `string`  | ‚ö†Ô∏è Optional | &#x2003;&emsp;`"sansaar_1"` (HQ) or `"sansaar_1_lite"` (Faster). |
| **`loop`**     | `boolean` | ‚ö†Ô∏è Optional | &#x2003;&emsp;&emsp;Whether the music should loop.               |

#### Code Samples

<Tabs>
  <Tab title="python" >
  
  ```python
    import requests

    url = "https://api.soundverse.ai/v1/generate/text-to-music"
    headers = {
        "Authorization": "Bearer your_api_key_here",
        "Content-Type": "application/json"
    }
    payload = {
    "userId": "61b7ca2f20e711017fc1b6c4",
    "prompt": "Compose a futuristic electronic track with cutting-edge synth sounds and pulsing beats.",
    "duration": 15,
    "model": "sansaar_1_lite",
    "loop": false
    }

    response = requests.post(url, json=payload, headers=headers)
    print(response.json())
    ```

   </Tab>

   <Tab title="curl">
  
  ```sh
    curl -X POST "https://api.soundverse.ai/v1/generate/text-to-music" \
    -H "Authorization: Bearer your_api_key_here" \
    -H "Content-Type: application/json" \
    -d '{
    "userId": "61b7ca2f20e711017fc1b6c4",
    "prompt": "Compose a futuristic electronic track with cutting-edge synth sounds and pulsing beats.",
    "duration": 15,
    "model": "sansaar_1_lite",
    "loop": false
    }'
    ```
