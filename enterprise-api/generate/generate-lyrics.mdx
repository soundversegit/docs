## Generate Lyrics from Prompt

This endpoint generates **AI-powered song lyrics** based on a **text prompt**.  
You can also specify:  
üé≠ **Mood** ‚Üí Set the emotional tone (e.g., happy, sad, energetic).  
üé∂ **Genre** ‚Üí Influence the lyrics with a musical style (e.g., pop, rock).  
üìñ **Section** ‚Üí Generate a specific part (e.g., verse, chorus).

---

#### **Core Parameters**

> üõà The parameters below define the request body for this API.

| Parameter     | Type     | Required?   | Description                                                           |
| ------------- | -------- | ----------- | --------------------------------------------------------------------- |
| **`prompt`**  | `string` | ‚úÖ Required | &#x2003;&emsp;&emsp;Text description of the lyrics.                   |
| **`mood`**    | `string` | ‚ö†Ô∏è Optional | &#x2003;&emsp;&emsp;Emotional tone of the lyrics.                     |
| **`section`** | `string` | ‚ö†Ô∏è Optional | &#x2003;&emsp;&emsp;Specify part of the song (verse, chorus, bridge). |
| **`genre`**   | `string` | ‚ö†Ô∏è Optional | &#x2003;&emsp;&emsp;Musical genre (pop, rock, hip-hop, etc.).         |

#### Code Samples

<Tabs>
  <Tab title="Python">
  
  ```python

    import requests

    url = "https://api.soundverse.ai/v1/generate/lyrics"
    headers = {
        "Authorization": "Bearer your_api_key_here",
        "Content-Type": "application/json"
    }
    payload = {
    "prompt": "string",
    "mood": "string" ,
    "section": "optional",
    "genre": "string"
    }



    response = requests.post(url, json=payload, headers=headers)
    print(response.json())
    ```

   </Tab>

   <Tab title="cURL">
  
  ```sh

curl -X POST "https://api.soundverse.ai/v1/generate/lyrics" \
 -H "Authorization: Bearer your_api_key_here" \
 -H "Content-Type: application/json" \
 -d '{
"prompt": "string",
"mood": "string" ,
"section": "optional",
"genre": "string"
}'

    ```

   </Tab>
