## Generate Lyrics from Prompt

This endpoint generates **AI-powered song lyrics** based on a **text prompt**.  
You can also specify:  
üé≠ **Mood** ‚Üí Set the emotional tone (e.g., happy, sad, energetic).  
üé∂ **Genre** ‚Üí Influence the lyrics with a musical style (e.g., pop, rock).  
üìñ **Section** ‚Üí Generate a specific part (e.g., verse, chorus).

#### Code Samples

<Tabs>
  <Tab title="Python">
  
  ```python

    import requests

    url = "https://api.soundverse.ai/v1/generate/lyrics"
    headers = {
        "Authorization": "Bearer your_api_key_here",
        "Content-Type": "application/json"
    }
    payload = {
    "userId": "string",
    "prompt": "string",
    "mood": "string" ,
    "section": "optional",
    "genre": "string"
    }



    response = requests.post(url, json=payload, headers=headers)
    print(response.json())
    ```

   </Tab>

   <Tab title="cURL">
