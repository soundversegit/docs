## üìÑ **Field Descriptions**

| Field                       | Type           | Description                                   |
| --------------------------- | -------------- | --------------------------------------------- |
| **\_id**                    | `UUID`         | Unique identifier for the generated response. |
| **userId**                  | `UUID`         | User ID of the requester.                     |
| **author**                  | `string/null`  | Author of the generated content.              |
| **prompt**                  | `string`       | The input prompt used to generate content.    |
| **projectId**               | `UUID`         | Associated project ID.                        |
| **createdAt**               | `ISO 8601`     | Timestamp when the response was created.      |
| **updatedAt**               | `ISO 8601`     | Timestamp of the last update.                 |
| **albumArt**                | `URL`          | Link to the album artwork.                    |
| **aiMagicTool.user**        | `string`       | User's AI input.                              |
| **aiMagicTool.assistant**   | `string`       | AI's generated response.                      |
| **aiReply**                 | `string`       | Final AI-generated reply.                     |
| **status**                  | `string`       | Status of the request.                        |
| **audioData**               | `array`        | Array containing generated audio details.     |
| **audioUrl**                | `URL`          | Link to the generated audio.                  |
| **bpm**                     | `integer/null` | Beats per minute (if applicable).             |
| **duration**                | `float/null`   | Length of the audio in seconds.               |
| **songName**                | `string/null`  | Name of the generated song.                   |
| **likeCount**               | `integer`      | Number of likes.                              |
| **playCount**               | `integer`      | Number of times played.                       |
| **audioType**               | `string`       | Type/category of audio.                       |
| **isLoopable**              | `boolean`      | Whether the audio can be looped.              |
| **isLiked**                 | `boolean`      | Whether the user liked the audio.             |
| **isHidden**                | `boolean`      | Whether the audio is hidden.                  |
| **muted**                   | `boolean`      | Whether the audio is muted.                   |
| **licenses**                | `string/null`  | Licensing details.                            |
| **isThread**                | `boolean`      | Whether it is part of a thread.               |
| **isPublic**                | `boolean`      | Whether the audio is public.                  |
| **isRestricted**            | `boolean`      | Whether access is restricted.                 |
| **loopedComponentUrl**      | `URL/null`     | URL of a looped version.                      |
| **loopedComponentDuration** | `float/null`   | Duration of looped segment.                   |
| **isStaffPick**             | `boolean`      | Whether it is a staff pick.                   |
