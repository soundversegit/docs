## 📌 Check the Status of a Generation Task

Use this API to retrieve the status of a generation task using `message_id`.

---

## 🔹 **Core Parameters**

> 🛈 The parameters below define the request for this API.

| Parameter        | Type     | Required?   | Description                                              |
| ---------------- | -------- | ----------- | -------------------------------------------------------- |
| **`message_id`** | `string` | ✅ Required | &#x2003;&emsp;&emsp;Unique ID of the generated response. |

---

## 🔍 **Request Format**

```http
GET  /status/generations/{message_id}
```

### 🖥️ Code Samples

<Tabs>
  <Tab title="Python">
  
  ```python
    import requests

    message_id = "your_message_id"
    url = f"https://api.soundverse.ai/status/generations/{message_id}"
    headers = {
        "Authorization": "Bearer your_api_key_here"
    }

    response = requests.get(url, headers=headers)
    print(response.json())

    ```

   </Tab>

   <Tab title="cURL">
  
  ```sh
   curl "https://api.soundverse.ai/status/generations/your_message_id" \
     -H "Authorization: Bearer your_api_key_here"
    ```
   </Tab>
   
</Tabs>
